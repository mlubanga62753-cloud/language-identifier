# Language Identification System (Swahili, English, Sheng, Luhya)

## Overview
This project detects the language of short text inputs using Machine Learning.

## Languages Supported
- Swahili
- English
- Sheng
- Luhya
- Code-mixed text detection

## Models Used
- Logistic Regression
- Support Vector Machine (SVM)

## Feature Extraction
- TF-IDF (Character n-grams)

## Deployment
Built using Streamlit and deployed online.
